# ✈️ SQL-задачи для анализа авиаперевозок (Demo Air)

Этот репозиторий содержит решения SQL-задач, выполненных в рамках учебного курса по анализу данных. В качестве источника данных используется демонстрационная база данных авиакомпании (схема `prod`), содержащая информацию о самолётах, аэропортах, рейсах, билетах и бронированиях.

## 📁 Структура репозитория

- `Avio_week_1.sql` – решения задач первой недели (базовые SELECT, фильтрация, сортировка, агрегатные функции, работа с JSON).
- `Avio_week_2.sql` – решения задач второй недели (JOIN, UNION, подзапросы, CASE, работа с датами, оконные функции и др.).
- `README.md` – этот файл.

## 📊 Используемые таблицы

В базе данных `prod` присутствуют следующие таблицы:

- `airplanes_data` – данные о самолётах (модель, дальность, скорость и т.д.)
- `airports_data` – справочник аэропортов (код, название, город, часовой пояс)
- `boarding_passes` – посадочные талоны
- `bookings` – бронирования (сумма, дата)
- `flights` – рейсы (статус, расписание)
- `routes` – маршруты (аэропорты вылета/прилёта, продолжительность, дни недели)
- `seats` – места в самолётах (класс обслуживания)
- `segments` – сегменты перелётов (цена, класс)
- `tickets` – билеты (пассажир, направление)
- `timetable` – представление с локальным временем вылета

## 🚀 Как запустить скрипты

### Требования
- PostgreSQL версии 12 или выше.
- Демо-база данных, развёрнутая в схеме `prod` (например, учебная БД «Авиаперевозки»).

### Запуск
Для проверки отдельных задач можно раскомментировать промежуточные `SELECT` (они закомментированы как подсказки).
